# clinicaIW
## Compilación
1. Ver script de base de datos y ejecutarlo para crearla
2. Entrar en el proyecto clinica-servidor
   1. npm i
   2. node app.js
3. Entrar en el proyecto clinica-cliente
   1. npm i
   2. npm run serve
   
> Ambos proyectos tienen que estar ejecutándose a la vez
